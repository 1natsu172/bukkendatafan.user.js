# 物件データファン.user.js
[物件ファン](https://bukkenfan.jp/)のトップページの各記事に物件の賃料や所在地などの情報を表示するユーザースクリプト

`https://bukkenfan.jp/`**トップページでのみ**動作します。記事を開く前に家賃とか駅近かなどの情報を知りたい人は使うと便利かもしれません。
